#### 青果教务教务系统爬虫



##### 实现概述

基于requests库实现网页得模拟请求，bs4进行网页数据内容的解析，实现网页数据的爬取。

##### 功能

* [x] 成绩查询

* [x] 课表查询

* [x] 考试安排查询

#### 参数说明（具体可查看浏览器所带参数）

| 参数名  | 含义                                       |
| ------- | ------------------------------------------ |
| hostUrl | 教务系统主页                               |
| host    | 请求的服务器域名                           |
| code    | 青果教务系统中学校代码                     |
| origin  | origin主要是用来说明最初请求是从哪里发起的 |
